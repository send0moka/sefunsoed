This is what the React component would look like:

<PodcastBlock
  title="Listen to Our Podcasts on Spotify"
  spotifyUrl="https://open.spotify.com/show/7MoeAljOnuz3QVMjptb5TQ?si=9zJzTPRUQMutRcUvp4KJiQ"
  useCustomPlayer={true}
  episode={{
    title: "Sample Podcast Episode",
    description: "This is a sample podcast episode description.",
    publishedDate: "2024-01-15",
    coverImage: "/media/image-hero1.webp",
    audioFile: "/media/sample-podcast.mp3",
    duration: 1800
  }}
/>

Usage Instructions:

  1. Add an MP3 file to /public/media/ folder
  2. In Payload CMS, add a new Podcast Block to any page
  3. Choose between Spotify embed or custom player
  4. For custom player: fill in episode details and audio file path
  5. For Spotify embed: paste the embed iframe code
